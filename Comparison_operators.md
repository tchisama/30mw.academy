



# comparison operators in rust 



## what are comparison operators ;


compearison operators are used for comparing the values of two operands.


exapmle : 

```rust
fn main() {
    let a = 10;
    let b = 20;
    println!("{}", a > b); // false
    println!("{}", a < b); // true
    println!("{}", a == b); // false
    println!("{}", a != b); // true
    println!("{}", a >= b); // false
    println!("{}", a <= b); // true
}

```
